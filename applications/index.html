Applications connectés

Le but des applications connectées est de récolter les données des appareils connectés, de les traiter et d’indiquer de façon simple à l’utilisateur comment améliorer sa santé à partir de ces dernières.

Récolte des données

Pour récolter des données, l’application a deux possibilités :

  • Récupérer les données depuis des appareils connectés tels que des montres connectés, des capteurs etc…
  • Créer des données à partir de l’appareil dont l’application est installé. Par exemple, une application de sommeil installé sur un portable pourra utiliser le microphone, une application de course sur une montre pourra utiliser la puce GPS et l'accéléromètre etc… Pour en savoir plus sur cet aspect de la collecte des données, nous vous invitons à voir le fonctionnement des objets connectés.

La récupération des données depuis des appareils connectés peut se faire de plusieurs façons : connexion Bluetooth, Wifi, cable USB, réseaux mobiles, etc… Ici, nous détaillerons le type de connexion le plus courant suite à un coût faible, une simplicité d’utilisation et de mise en place et une efficacité suffisante : le Bluetooth.
